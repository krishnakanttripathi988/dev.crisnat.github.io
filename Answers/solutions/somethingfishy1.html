Converting datatype of all the elements present in the given Dataframe.

How to convert all the elements of given dataframe to a specified datatype

Solution is given below:

How to type cast dataframe columns in Python


Here we are about to learn that how we can change the datatype of a given dataframe or in the simple words how to type cast a dataframe.


using "astype"


First of all let's learn about the syntax:

Let's see this simple piece of code:


df = pandas.Dataframe()
df.astype('int64')


Here all the values will be type casted to the datatype "int64"


we can have all the datatype as the attributes of astype like:

  • For String we use str
  • For float we use float
  • For integer we use int

using "lambda"


First of all let's learn about the syntax:

Let's see this simple piece of code:


df = pandas.Dataframe()
df["any column name"] = df.apply(lambda x : (int64)(x))


Here all the values will be type casted to the datatype "int64"


By Iterating


First of all let's learn about the syntax:

Let's see this simple piece of code:


df = pandas.Dataframe()
for i in df[" any column name"]:
    i = (int64)(i)


Here all the values will be type casted to the datatype "int64"
